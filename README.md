### kontribusi project

buat kalian yang ingin berkontribusi dengan project ini silahkan instal di komputer kalian.

- Instal project 
```md
https://github.com/progate-nextlevel/inventory.git
```
- Instal composer 
```md
composer install
```
- Duplicat file `.env`
 ```md 
 cp .env.example .env
 ```
- jalankan server
```md
php artisan serve
```
